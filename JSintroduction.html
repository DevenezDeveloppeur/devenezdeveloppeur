Pourquoi JavaScript ?

Le HTML, le CSS et le JS représentent les socles du développement de pages Web interactives. Ainsi, le Web tel qu’on le connait est basé sur trois piliers : le HTML, le CSS et le JavaScript.

  • HTML est le langage de balisage permettant de structurer et donner du sens à notre contenu Web (définir des paragraphes, des titres, des tables, des images …)
  • CSS est le langage qui nous permet de mettre en forme nos pages Web et styliser notre contenu HTML. En effet, Les feuilles de style, contiennent du code qui permet de gérer le design d’une page en HTML, comme par exemple définir des couleurs, des polices de caractères, bien disposer nos éléments HTML, etc.
  • JavaScript est le langage qui permet de rendre un document HTML interactif/dynamique. En outre, JavaScript est un vrai langage de programmation qui permet de créer des algorithmes.

Ce qui rend JS un langage en grande demande, c’est le fait qu’il est :

  • Dynamique : Il implémente plusieurs éléments d’un haut niveau de complexité sur des pages Web, tels que des animations, des galeries d’images, des variations de mises en page. Avec JS il est possible de créer des jeux, des graphiques 2D, des animations 3D animées et bien plus encore chose qui rend l’expérience Utilisateur beaucoup plus intuitive et agréable ;
  • Polyvalent : Il permet de développer la partie front-end (le client : le coté visible) et/ou le développement back-end (bases de données, API, authentification, le serveur, etc : le coté invisible) ;
  • Flexible: fonctionne dans différents environnements (les smart TV, IoT, les applications mobiles, les applications bureau multiplateformes, les applications en réalité virtuelle, les jeux, des extensions de navigateurs, la géolocalisation…).
  • Populaire : Dispose d’une large communauté de développeurs extrêmement active, réceptive et ouverte à tous ceux qui souhaitent s’y aventurer, et une multitude de forums et de services qui mettent des ressources gratuites à disposition des développeurs ;
  • Idéal pour débuter en programmation et Accessible : Etant installé sur tous les navigateurs web (et d’ailleurs c’est le seul langage), il s’avère le meilleur langage web avec lequel il est facile de commencer le développement web; Etant un langage de programmation à part entière, Orienté Objet, interprété, de haut niveau, JS permet d’apprendre les bases de programmation, d’élaborer des algorithmes basiques mais aussi des solutions de haute complexité ;
  • Un gros potentiel de carrière en tant que développeur Web et Mobile: trop demandé dans le marché du travail, Etant classé 3ème parmi les langages les populaires en 2021 après python et JAVA ;

Historique :

Le JavaScript est le premier langage de script créé pour le Web. Il a été mis au point par Netscape en 1995. A l'origine, il se nommait LiveScript et était destiné à fournir un langage de script simple au navigateur Netscape Navigator 2.

Depuis sa création en 1995, le langage qui est finalement connu sous le nom de JavaScript a connu plusieurs itérations et versions.

JavaScript a été inventé par Brendan Eich et est devenu en 1997 une norme ECMA (European Computer Manufactures Association). ECMAScript est le nom de la langue officielle. Les versions ECMAScript incluent ES1, ES2, ES3, ES5 et ES6…ES9(2018).

Javascript a parcouru un long chemin, c'est-à-dire d'une programmation/Framework côté client (comme Angular/React) à une technologie côté serveur comme NodeJS en 2009.

Ou écrire le code ?

  • Le code JS doit être incorporé dans la balise <script> </script>;
  • Le code peut être placé dans la balise <head> ou dans le <body> avant juste sa fermeture ;
  • Le script peut être placé dans un fichier externe .js afin de séparer le code HTML du JS et permettre sa réutilisation plusieurs fois sans avoir à le réécrire. Pour l’importer il suffit d’utiliser
    <script src="myScript.js"> </script>;
